二、基础模型（无成本差异）

核心结论：无论「表面积固定求最大体积」还是「体积固定求最小表面积」，最优解均为 h=2R（高是半径的2倍）

方案一：表面积固定 → 体积最大（核心目标：最大化容积）

已知条件：圆柱表面积 S₀ = 2πR² + 2πRh（固定值）

步骤1：建立体积函数（单变量）

由表面积公式变形求 h：h = (S₀ - 2πR²) / (2πR)

代入体积公式：V(R) = πR² × [(S₀ - 2πR²) / (2πR)] = (S₀R - 2πR³) / 2 = (S₀/2)R - πR³

步骤2：对R求导（找极值点）

V’(R) = S₀/2 - 3πR²

步骤3：令导数=0，求最优半径

令 V’(R) = 0 → S₀/2 - 3πR² = 0 → R² = S₀/(6π) → R = √[S₀/(6π)]

步骤4：求最优高h，推导高径比

将R代入h的表达式：h = (S₀ - 2π×S₀/(6π)) / (2π×√[S₀/(6π)]) = (2S₀/3) / (2π×√[S₀/(6π)]) = √[S₀/(6π)] × 2

最终得：h = 2R（高是半径的2倍，高径比2:1）

结论：表面积固定时，h=2R时体积最大（无成本差异）

方案二：体积固定 → 表面积最小（核心目标：最省材料）

已知条件：圆柱体积 V₀ = πR²h（固定值）

步骤1：建立表面积函数（单变量）

由体积公式变形求 h：h = V₀/(πR²)

代入表面积公式：S(R) = 2πR² + 2πR×[V₀/(πR²)] = 2πR² + 2V₀/R

步骤2：对R求导（找极值点）

S’(R) = 4πR - 2V₀/R²

步骤3：令导数=0，求最优半径

令 S’(R) = 0 → 4πR = 2V₀/R² → 4πR³ = 2V₀ → R³ = V₀/(2π) → R = ∛[V₀/(2π)]

步骤4：求最优高h，推导高径比

将R代入h的表达式：h = V₀/(π×[V₀/(2π)]) = 2∛[V₀/(2π)] = 2R

最终得：h = 2R（高是半径的2倍，高径比2:1）

结论：体积固定时，h=2R时表面积最小（无成本差异）

三、成本差异优化后计算（瓶身:瓶底=1:1.5）

比例优化过程：

✅ 基础模型（无成本差异）：h = 2R（高:半径=2:1）

✅ 成本差异优化（加权表面积）：

加权表面积 S加权 = 1.5×2πR² + 1×2πRh = 3πR² + 2πRh

按上述相同步骤求导后可得：h = 3R（高:半径=3:1）→ 符合实际可乐罐设计！

体积固定 → 加权表面积最小：h = 3R

表面积固定 → 体积最大：h = 3R

即：考虑成本差异后，圆柱的高 = 3倍底面半径时，利润最大化！

分层练习

基础题

1. 一个圆柱体积为500cm³，考虑瓶身/瓶底成本1:1.5，求用料最省时的底面半径和高。

参考答案：
R = ∛(500/(3π)) ≈ 3.75 cm
h = 3R ≈ 11.25 cm

2. 圆柱表面积固定400cm²，考虑成本差异，求最大容积和对应利润。

参考答案：
R = √(400/(8π)) ≈ 3.99 cm
h = 3R ≈ 11.97 cm
V ≈ π×(3.99)²×11.97 ≈ 598 cm³

实践题

测量身边的饮料罐，计算高径比 h/R，是否接近3？对比理论值分析差异原因。

参考答案：
真实可乐罐高径比≈3.18，略大于理论值3，原因：
1. 罐底/罐盖更厚，成本比略大于1.5
2. 握持手感、装箱稳定性要求
3. 生产工艺限制

模型优化：为什么真实可乐罐不完全符合 h=3R？

  • 瓶底、瓶盖厚度不同（成本差异进一步放大）
  • 人体工学握持（高径比需兼顾手感）
  • 装箱、运输稳定性（高径比过大会增加破损率）
  • 生产工艺限制（模具标准化、成型难度）
  • 市场消费习惯（视觉审美、便携性）

利润模型总结：

1. 基础模型（无成本差异）：利润 = (V - S) × 0.1 → 最优比 2:1

2. 成本优化模型（瓶身:瓶底=1:1.5）：利润 = (V - (1.5×2πR² + 1×2πRh)) × 0.1 → 最优比 3:1

3. 工程落地：3.18:1（综合工艺、运输、成本后的实际最优解）

四、结构设计特点分析

  • 非等厚设计：罐身薄、盖底厚，成本系数1:1.5，完美匹配数学模型的成本差异假设，是利润最大化的核心设计。
  • 凸形罐底：采用球面凸底结构，将内压转化为环向应力，大幅提升耐压能力，相比平底可减少30%以上的材料使用。
  • 渐变罐肩：罐身到罐口的锥形过渡设计，既满足握持手感，又分散灌装、运输过程中的应力集中。
  • 标准化直径：66mm直径是综合考虑人体握持尺寸、装箱效率、生产线兼容性的最优选择。
  • 高径比优化：3.18的高径比虽偏离数学最优值3，但结合仓储堆码、运输成本、消费习惯等工程因素，是全生命周期成本最低的选择。
  • 材料选型：3004铝合金（罐身）兼顾成形性和耐腐蚀性，5182铝合金（罐盖）具备良好的撕裂性能，实现功能分区优化。
